API Document

1. Đăng nhập (Login) auth login
POST /api/login
Body (JSON):
{
    "username": "your_username",
    "password": "your_password"
}
Response:
{
    "token": "your_jwt_token",
    "name": "your_username"
}
2. Lấy danh sách sản phẩm product list
GET /api/products
Headers:
Authorization: Bearer <token>
Response:
[
    {
        "id": 1,
        "name": "Key Tool Pro",
        "type": "keytool",
        "price": 15000,
        "status": "available"
    }
]
3. Mua sản phẩm order product
POST /api/order
Headers:
Authorization: Bearer <token>
Body (JSON):
{
    "product_id": 1,
    "quantity": 1
}
Response:
{
    "order_id": 123,
    "status": "success",
    "key": "xxxx-xxxx-xxxx"
}
