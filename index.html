Nguồn cung cấp tài nguyên MMO

Hỗ trợ người làm MMO với bộ sưu tập công cụ đa dạng: từ tài khoản mạng xã hội, tool tiện ích, đến license hỗ trợ tối ưu quy trình kiếm tiền online.

Dịch vụ API đa dạng, phong phú

Nền tảng cung cấp dịch vụ API đa dạng, dễ tích hợp – từ xác thực, xử lý dữ liệu, thanh toán đến mạng xã hội. Giúp bạn xây dựng ứng dụng nhanh chóng, ổn định và linh hoạt và vô vàn dịch vụ API khác sẳn sàng hoạt động.

Tích hợp công cụ

Tích hợp hàng loạt công cụ phục vụ MMO – từ automation, tool social, airdrop, đến phần mềm hỗ trợ tối ưu hiệu suất kiếm tiền online. Giao diện dễ dùng, cập nhật liên tục.

API Document

1. Đăng nhập (Login) auth login
POST /api/login
Body (JSON):
{
    "username": "your_username",
    "password": "your_password"
}
Response:
{
    "token": "your_jwt_token",
    "name": "your_username"
}
2. Lấy danh sách sản phẩm product list
GET /api/products
Headers:
Authorization: Bearer <token>
Response:
[
    {
        "id": 1,
        "name": "Key Tool Pro",
        "type": "keytool",
        "price": 15000,
        "status": "available"
    }
]
3. Mua sản phẩm order product
POST /api/order
Headers:
Authorization: Bearer <token>
Body (JSON):
{
    "product_id": 1,
    "quantity": 1
}
Response:
{
    "order_id": 123,
    "status": "success",
    "key": "xxxx-xxxx-xxxx"
}
